PWEBATIVIDADE10 - Calculadora de IMC

Projeto desenvolvido para a disciplina de Programação Web.

## Objetivo

Criar uma aplicação web para calcular e mostrar o IMC de uma pessoa.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Entrada da altura
- Entrada do peso atual
- Cálculo do IMC
- Exibição da classificação
- Exibição do grau de obesidade
- Uso de funções em JavaScript
- JavaScript em arquivo externo
